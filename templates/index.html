Pairs Trading with Machine Learning

This project uses Machine Learning methods to group assets based on similar factor loadings, then identifies pairs within each cluster to implement pairs trading strategy. Pairs are then constructed into a market-neutral portfolio.

Machine Learning Methods

Apply Principal Component Analysis(PCA) to reduce the dimension of the returns data and factor(Indsutry) data, then group stocks using DBSCAN clustering.

Finding Pairs

Run cointegration test(ADF test on stationarity of residual) on each pair in each cluster. Get the most cointegrated pair.

Trading Logic

Trade on residuals of pair prices. Then apply bollinger band strategy: If the residual just crosses below lower band, long; if the residual crosses above higher band, short; otherwise, hold on to current position.
